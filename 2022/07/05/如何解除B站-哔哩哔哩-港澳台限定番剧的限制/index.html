如何解除B站(哔哩哔哩)港澳台限定番剧的限制

爱好B站番剧的同学，大概遇到过 bilibili 的提示”非常抱歉，根据版权方要求，您所在的地区无法观看本片～“。即便你开通了大会员，这些仅港澳台限定番剧依然无法观看。

1. 解决办法

对于浏览器，我们可以使用 JS 脚本，通过替换获取视频地址接口的方式，实现解除B站区域限制，在此感谢作者 ipcjs 提供的开源脚本。

  1. 这里额外说一下，身居海外 的朋友们也可以通过此方法解除某些视频仅限大陆观看的限制，只需要在下面教程的设置中，添加大陆解析接口。

2. 安装脚本

2.1 浏览器需：

Chrome 谷歌浏览器（下载） 或 Windows 自带的 Edge 浏览器，其他游览器自己测试。

以下是Chrome 谷歌浏览器的教程， Edge 浏览器应该是比较好安装的，具体安装可以参考下面的教程，不行自行百度。

2.2 安装插件：

在 https://crxdl.com/ 搜 dhdgffkkebhmkfjojejmpbldmpobfkfo 下载油猴插件，并将下载到的压缩包解压。

没错万能的油猴，Edge 浏览器可以自己去插件商店搜索油猴。

解压包里面有个叫crx的文件夹，文件夹里面有个叫 dhdgffkkebhmkfjojejmpbldmpobfkfo_v4.16.1.crx 的文件，把他拖到游览器里面去，QQ游览器，360游览器，不一定能用 没试过这两个游览器，火狐自行测试。

在 Chrome 浏览器地址栏输入 chrome://extensions/ 打开，启动右上角的开发者模式，将解压后的 .crx 文件拖动到此页面完成安装。

2.3 添加脚本：

在 https://wanji.app/js/875.html 点击最新版脚本进入安装页面，完成脚本的安装。

会让关注微信获取验证码，验证码为：1219 不一定正确，不对自己加关注获取。

2.4 启用脚本

以番剧《迦希大人不气馁》为例，先打开网址：https://www.bilibili.com/bangumi/media/md28234736 然后点击浏览器右上角的扩展程序图标->点击油猴插件右侧钉子图标->点击油猴插件图标->启用油猴插件和启用解除限制脚本

3. 设置脚本

脚本启用后，点击《迦希大人不气馁》，刷新页面，会增加一个如图所示的星球图标，点击即可进入脚本设置页面。

第一步，就是点击左下角的 账号授权 选项，只有授权了账号信息后，才能获取更高的视频分辨率，以及你开通的大会员信息。

第二步，测试是否已经可以观看港澳台限定番剧了，如果依然无法观看，那么有可能解析视频的接口失效了，你可以更换为下方的一些共享接口。

4. 哔哩哔哩解析接口

  • 注意解析接口有区域性，有些接口只能解析香港，有的只能解析台湾的，当然有些可以大陆、港澳台一并解析
  • 注意解析接口有时效性，有些接口的服务器到期，可以尝试复制接口网址到浏览器打开，能打开的多半还活着

使用手机与电视观看哔哩哔哩(B站)港澳台限定番剧

日常都是使用手机看B站港澳台限定地区的番剧，并且手机哔哩哔哩客户端支持投屏到电视观看，体验非常好。使用手机的同学请阅读本教程，使用电脑的同学请前往《电脑观看B站港澳台限定番剧限制》。

安装哔哩漫游APP

哔哩漫游是B站的修改版，内置了解除B站番剧区域限制、港澳台 CDN 加速、缓存番剧、关闭青少年模式弹窗和自动点赞等增强功能，由 LoveSy 开源于 GitHub

  • 注意：安装前需先卸载已安装的官方版
  • 下载：[哔哩漫游APP](xpatch版 (lanzoui.com)) 密码：2333

设置港澳台番剧解析

设置过程

打开哔哩漫游➡登录账号➡我的➡哔哩漫游设置➡点击启动解锁番剧限制➡设置解析服务器➡香港、台湾、东南亚填写 api.qiu.moe ➡确定➡确定并重启

更多B站解析接口

注意，手机端添加解析接口时不要擅自加 https 或 http ，直接复制粘贴下方的任意接口就行。

如何搜索港澳台番剧

搜索关注 哔哩哔哩番剧出差 账号，B站所有港澳台限定番剧都由这个账户发布。

如何在电视上看番剧

先为电视或智能电视盒子安装B站电视客户端云视听小电视，然后打开哔哩漫游或哔哩哔哩APP，点我的，在屏幕右上角有个投屏图标，点击后可投屏到电视播放，并且还能显示弹幕！
